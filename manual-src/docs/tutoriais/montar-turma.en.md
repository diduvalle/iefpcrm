# Guided tutorial: set up the class in 10 minutes

!!! abstract "In this tutorial"
    **Goal:** create the class, import the trainees, share access and be ready to receive submissions. · **Duration:** ~10 min · **For:** Trainer.

## Before you start

- [x] You have your **IEFP institutional email** (`forNNNN@formacao.iefp.pt`).
- [x] You have the **list of trainees** (first name, surname, email).

---

## Step 1 · Create the class

1. Open **[iefpcrm.cr0x.org](https://iefpcrm.cr0x.org)** → **“I’m a trainer — create a new class”**.
2. Fill in IEFP email, name, **class number (8 digits)**, username and password → **Create class and enter**.

!!! warning "Save the recovery code"
    It appears **once** — write it down. It is your way to recover the password.

!!! note "📷 Screenshot"
    ![Home screen — create a class](../assets/screens/hub.png)

---

## Step 2 · Prepare the list (Excel)

1. **Settings → Class trainees → Excel/CSV Template**.
2. Fill in one row per trainee: `first name · surname · email · username · password`.

!!! tip "Password"
    If you leave the `password` column empty, the **initial password** is used (field on the page) — the same for everyone; each trainee changes it later.

---

## Step 3 · Import with preview

1. Click **Import list** and choose the file.
2. Check the **preview**: **OK** rows in green, errors flagged in red.
3. **Import N valid**.

!!! success "Result"
    The trainees appear in the list, ready to enter.

!!! tip "Preview"
    Before saving, you see a table with each row marked **✓ valid** or in **red** (missing field, invalid or duplicate email). Only the valid rows are imported.

---

## Step 4 · Share access

Send the class the link with the number:

```
https://iefpcrm.cr0x.org/?t=12345678
```

Each trainee enters with **their own username and password**.

---

## Step 5 · (Optional) Enable real email sending

In **Settings → Real email sending**, turn on **Enable real sending** and run **Send test**. *(The keys are fixed; you only manage the mode.)*

---
