# FAQ — perguntas frequentes

??? question "Esqueci-me da palavra-passe. E agora?"
    **Formando:** pede ao formador para a repor (*Gerir a turma*). **Formador:** usa o **código de recuperação** que recebeste ao criar a turma.

??? question "Escrevi o número da turma e diz que não existe."
    Confirma o número (8 dígitos) com o formador. Se a turma existir, entras; senão, a app avisa para evitar enganos.

??? question "Os meus dados desaparecem se fechar o browser?"
    Não. A tua *sandbox* fica guardada **no teu computador** (neste browser). Mudar de computador ou limpar os dados do browser apaga-a — por isso submete o trabalho quando estiver pronto.

??? question "Posso submeter o trabalho mais do que uma vez?"
    Sim, até **2 vezes**. À terceira fica bloqueado. Submete só quando estiveres mesmo pronto.

??? question "Porque é que a app pede uma assinatura para submeter?"
    A assinatura **autentica a autoria** do trabalho. Defines-a uma vez no teu perfil.

??? question "O email de teste chegou ao spam / diz 'remetente externo'."
    É normal — os emails saem por um serviço externo. O **nome** é IEFP CRM e as **respostas** vão para o email certo. Para envios a clientes reais, isso é inofensivo.

??? question "O formador consegue ver o meu trabalho?"
    Sim, quando **submetes**. Ele vê os teus clientes e propostas (só leitura) e pode gerar o PDF das tuas propostas. Antes de submeteres, ninguém vê a tua *sandbox*.

??? question "Como reponho os dados de exemplo?"
    Em **Definições → Dados & backup**. Também podes **gerar mais exemplos** (clientes, empresas, propostas) no cartão respetivo.
