# Pipeline (Kanban)

As propostas organizam-se por **estado**, em colunas que podes arrastar.

`Lead → Qualificação → Negociação → Ganha → Perdida`

## Como funciona

- **Arrastar** um cartão entre colunas muda o estado (regista quem e quando).
- Cada coluna mostra o **total** ao fundo.
- Largar em **Perdida** pede o **[motivo da perda](perda.md)**.
- Propostas **Ganha/Perdida** ficam **bloqueadas** (só leitura).

## Reabrir

Numa proposta fechada, **Reabrir** volta a colocá-la em *Negociação* e desbloqueia as linhas.

!!! tip "Vista Lista"
    Preferes uma tabela? Alterna para **Lista** no topo do módulo — mesmos dados, outra leitura.
