# Exportar PDF

Gera o documento **A4** pronto a imprimir ou guardar.

## Passo a passo

1. Abre a proposta → barra de ações.
2. **Pré-visualizar** — vê a “folha” como vai sair.
3. **Descarregar PDF** → na janela de impressão do browser, escolhe **Guardar como PDF**.

O documento usa o **[modelo](../modelos.md)** da proposta (cabeçalho, logótipo, cotação, totais, condições, assinatura) e os **dados do cliente**. O rodapé fica fixo no fundo de cada página.

!!! tip "Trocar de modelo"
    Cada proposta pode usar um **modelo** diferente. Experimenta trocar e voltar a exportar para comparar.
