*Criar uma proposta, do cliente aos totais.*

Clica **+ Proposta** (ou num cartão existente para editar). O modal abre com o cabeçalho da proposta.

## Campo a campo

| Campo | Notas |
|---|---|
| **Número** | Automático (`PROP-{ano}-{nº}`), só leitura — reinicia a cada ano. |
| **Cliente** | Escolhe da lista de contactos. Define o **IVA por região** e o **desconto de loyalty**. |
| **Título** | Nome da proposta (ex.: *Licenciamento CRM*). |
| **Validade** | Data até à qual a proposta é válida. |
| **Estado** | Lead / Qualificação / Negociação / Ganha / Perdida. |

## Passo a passo

1. **Propostas → + Proposta**.
2. Escolhe o **cliente** (define IVA e desconto).
3. Preenche **título** e **validade**.
4. Adiciona as **[linhas do catálogo](linhas.md)**.
5. **Guardar**.

!!! tip "Auditoria"
    A proposta guarda **quem criou/alterou e quando** — visível no rodapé do modal.
