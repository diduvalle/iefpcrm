# Remover um formando

Tira o acesso de um formando à turma.

## Passo a passo

1. **Definições → Formandos da turma** — localiza o formando.
2. Clica **✕** na linha dele.
3. Confirma.

O **acesso deixa de funcionar** imediatamente.

!!! warning "Os dados locais não são afetados"
    Remover o acesso **não apaga** os dados de trabalho que estão no **computador do formando** (a *sandbox* é local). Só se remove o **acesso online** à turma.

!!! note "Não te podes remover a ti"
    Por segurança, não consegues remover a tua própria conta de administrador.
