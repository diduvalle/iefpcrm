# Adicionar um formando individual

Para criar um acesso a um, sem o ficheiro de importação.

## Passo a passo

1. **Definições → Formandos da turma → Adicionar individualmente**.
2. Preenche:
    - **Nome** *(obrigatório)* e **Apelido**.
    - **Email** do formando.
    - **Utilizador** — se deixares vazio, é **gerado automaticamente** a partir do nome (ex.: `joao.silva`).
    - **Palavra-passe** *(obrigatória)*.
    - **Papel** — Formando (predefinido) ou Formador.
3. **Adicionar à turma**.

O formando aparece na lista, pronto a entrar com o link `?t=…`.

!!! tip "Co-formador"
    Para teres um **co-formador**, escolhe o papel *Formador* — terá acesso de gestão à turma.
