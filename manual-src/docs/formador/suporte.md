# Suporte (root)

O **modo de suporte** dá acesso global à **identidade** (turmas e contas) — pensado para o responsável pela plataforma ajudar formadores (ex.: repor uma palavra-passe esquecida).

## Como entrar

1. No ecrã inicial (sem turma), abre o seletor **Turma** → **Entrar por código…**
2. Em vez de um número, escreve **`root`** (ou `suporte`).
3. Mete a **palavra-passe de suporte**.
4. Vês a **lista de todas as turmas** → **Abrir** entra como administrador dessa turma.

!!! note "Alcance do suporte"
    O suporte alcança **turmas, contas e palavras-passe** (a identidade). **Não** vê os dados de prática (clientes/propostas) que cada formando criou — esses são locais, por desenho.

## Recuperação de palavra-passe

- **Formando** esqueceu-se → o **formador** repõe em *Gerir a turma*.
- **Formador** esqueceu-se → usa o **código de recuperação** que recebeu ao criar a turma, ou pede ao **suporte (root)** para repor.

!!! warning "Segurança"
    A credencial de suporte é única e fica fora do código do browser. Mesmo assim, trata-a como uma chave-mestra — não a partilhes.
